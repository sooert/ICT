          - 범위
               - TypeScript(ms(제작), 구글(확산)) > ESNext > ES5
               - TypeScript -> ESNext -> ES5 -> 모든 브라우저 호환

                    - TS -> transcompile(컴파일) -> ESNext -> 처리(표준화) : babel -> ES5 생성 -> webpack -> 압축
                    - trans compile 처리하여 사용
